Formularios

Etiquetas para crear formularios interactivos. Permiten al usuario ingresar datos y enviarlos al servidor o procesarlos con JavaScript.

<form> Formularios
Definición

Contenedor que agrupa todos los campos del formulario. Define hacia dónde se enviarán los datos (action) y el método de envío (method).

Sintaxis básica
<!-- action: URL destino. method: "get" o "post" -->
<form action="/enviar" method="post">
  <!-- aquí van los campos -->
</form>
✦ Buenas prácticas

Usa method="post" para enviar datos sensibles (contraseñas, información personal). Usa method="get" para búsquedas donde quieras que los datos aparezcan en la URL.

<label> <input> Formularios
Definición

<label> — Etiqueta descriptiva que identifica a qué campo pertenece. Al hacer clic en el label, el campo correspondiente recibe el foco.
<input> — Campo de entrada de datos. Cambia completamente según el atributo type. Es una etiqueta vacía (sin cierre).

Tipos de input más usados
<!-- Texto libre -->
<input type="text">
<!-- Correo electrónico (valida el formato) -->
<input type="email">
<!-- Contraseña (oculta los caracteres) -->
<input type="password">
<!-- Número -->
<input type="number">
<!-- Fecha -->
<input type="date">
<!-- Casilla de verificación -->
<input type="checkbox">
<!-- Botón de opción (uno a la vez) -->
<input type="radio">
Cómo enlazar label con input
<!-- El "for" del label debe ser igual al "id" del input -->
<label for="nombre">Tu nombre:</label>
<input type="text" id="nombre" name="nombre" required>
✦ Buenas prácticas

Siempre vincula cada <label> con su campo usando los mismos valores en for e id. Usa required para campos obligatorios y placeholder para dar un ejemplo del valor esperado.

<textarea> <select> <button> <fieldset> <legend> Formularios
Definición

<textarea> — Área de texto de múltiples líneas para comentarios o mensajes largos.
<select> / <option> — Lista desplegable donde el usuario elige una opción.
<button> — Botón interactivo para enviar el formulario u otra acción.
<fieldset> / <legend> — Agrupa campos relacionados dentro del formulario y les pone un título visible.

Sintaxis básica
<!-- Área de texto: rows controla la altura -->
<textarea id="mensaje" name="mensaje" rows="4"></textarea>

<!-- Lista desplegable -->
<select name="nivel">
  <option value="basico">Básico</option>
  <option value="avanzado">Avanzado</option>
</select>

<!-- Botón de envío -->
<button type="submit">Enviar</button>

Datos personales
Preferencias
✦ Buenas prácticas

Usa <fieldset> para agrupar campos relacionados (datos personales, preferencias, etc.). Siempre especifica el type del <button>: type="submit" para enviar, type="reset" para limpiar, type="button" para acciones con JavaScript.
