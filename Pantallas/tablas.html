Tablas

Etiquetas para mostrar información organizada en filas y columnas. Ideal para datos comparativos, horarios, precios, resultados, etc.

<table> Tablas
Definición

Es el contenedor principal que envuelve toda la tabla. Dentro de él van el <thead> (cabecera) y el <tbody> (cuerpo con datos).

Sintaxis básica
<table>
  <!-- cabecera con nombres de columnas -->
  <thead> ... </thead>
  <!-- cuerpo con los datos -->
  <tbody> ... </tbody>
</table>
✦ Buenas prácticas

Las tablas son solo para datos tabulares, nunca para organizar el diseño visual de la página. Para maquetación usa CSS Flexbox o Grid.

<thead> <tbody> <tr> <th> <td> Tablas
Definición

<thead> — Agrupa las filas del encabezado de la tabla (nombres de columnas).
<tbody> — Agrupa las filas del cuerpo de la tabla (los datos reales).
<tr> — Una fila de la tabla (table row).
<th> — Celda de encabezado. Por defecto se muestra en negrita y centrada.
<td> — Celda de datos normal (table data).

Sintaxis básica
<table>

  <!-- thead: fila con los nombres de las columnas -->
  <thead>
    <tr>
      <th>Etiqueta</th>
      <th>Categoría</th>
    </tr>
  </thead>

  <!-- tbody: filas con los datos reales -->
  <tbody>
    <tr>
      <td>&lt;header&gt;</td>
      <td>Estructura</td>
    </tr>
  </tbody>

</table>

Uso adecuado

Usa <th> para los encabezados de columna o fila. Puedes añadir el atributo scope="col" para indicar que es encabezado de columna, o scope="row" para fila. Esto mejora la accesibilidad con lectores de pantalla.

✦ Buenas prácticas

Añade un <caption> justo después de <table> para describir el propósito de la tabla. Esto mejora la accesibilidad. No uses colspan ni rowspan de manera excesiva; si la tabla se vuelve muy compleja, considera dividirla en varias tablas simples.
