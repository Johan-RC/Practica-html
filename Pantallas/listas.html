Listas

Etiquetas para organizar información en grupos: con viñetas, numeradas, o en formato de glosario (término y definición).

<ul> <ol> <li> Listas
Definición

<ul> — Lista desordenada (unordered list). Muestra viñetas. El orden no importa.
<ol> — Lista ordenada (ordered list). Muestra números. El orden sí importa.
<li> — Elemento de lista (list item). Se usa dentro de <ul> o <ol>.

Sintaxis básica
<!-- Lista desordenada: cuando el orden no importa -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<!-- Lista ordenada: cuando el orden sí importa -->
<ol>
  <li>Escribir el HTML</li>
  <li>Agregar CSS</li>
  <li>Añadir JavaScript</li>
</ol>
Ejemplo práctico
<ul> — desordenada
  • HTML
  • CSS
  • JavaScript
<ol> — ordenada
  1. Escribir HTML
  2. Agregar CSS
  3. Añadir JS
Uso adecuado

Usa <ul> para colecciones de ítems sin jerarquía de orden (ingredientes, características). Usa <ol> para instrucciones, pasos o rankings donde el orden importa.

✦ Buenas prácticas

Solo pon <li> directamente dentro de <ul> u <ol>, nunca otros elementos directamente. Puedes anidar listas colocando un <ul> dentro de un <li> para crear sublistados.

<dl> <dt> <dd> Listas
Definición

<dl> — Lista de definiciones (description list). Funciona como un glosario.
<dt> — El término a definir (description term).
<dd> — La definición o descripción del término (description details).

Sintaxis básica
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>

  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>
Ejemplo práctico
HTML
Lenguaje de marcado para la web
CSS
Hojas de estilo en cascada
JS
JavaScript, lenguaje de programación web
Uso adecuado

Ideal para glosarios de términos técnicos, páginas de preguntas frecuentes (FAQ), fichas de productos con atributos, y cualquier contenido con pares de "nombre: valor".

✦ Buenas prácticas

Un <dt> puede tener más de un <dd> si el término tiene varias definiciones o acepciones. No uses <dl> solo por su aspecto visual; úsalo cuando realmente haya una relación de término-definición.
