Texto y Contenido

Etiquetas para organizar, jerarquizar y dar formato al texto. Desde los títulos más importantes hasta el énfasis en una sola palabra.

<h1> — <h6> Texto
Definición

Los encabezados crean jerarquía en el contenido. <h1> es el más importante (solo uno por página). <h6> es el de menor importancia. Son fundamentales para el SEO y la accesibilidad.

Sintaxis básica
<h1>Título principal (uno solo por página)</h1>
<h2>Sección</h2>
<h3>Subsección</h3>
<h4>Detalle</h4>
<h5>Menor detalle</h5>
<h6>Nivel mínimo</h6>
Ejemplo práctico

Encabezado h1

Encabezado h2

Encabezado h3

Encabezado h4

Encabezado h5

Encabezado h6

Uso adecuado

Úsalos para organizar el contenido en niveles. <h1> para el título de la página, <h2> para las secciones, <h3> para subsecciones, y así sucesivamente.

✦ Buenas prácticas

No saltes niveles (por ejemplo de h1 directo a h4). Usa los encabezados para estructurar el contenido, no para cambiar el tamaño del texto visualmente (para eso usa CSS).

<p> Texto
Definición

Define un párrafo de texto. Es uno de los elementos más usados en HTML. El navegador añade automáticamente un espacio antes y después del párrafo.

Sintaxis básica
<p>Este es el primer párrafo.</p>
<p>Este es el segundo párrafo.</p>
Ejemplo práctico

Este es el primer párrafo. Tiene algo de texto para ver cómo se ve en la página.

Este es el segundo párrafo. El navegador añade automáticamente un espacio entre ellos.

✦ Buenas prácticas

No uses <br> entre dos párrafos para crear espacio. En su lugar, usa la propiedad margin de CSS. Cada bloque de texto independiente debe ir en su propio <p>.

<strong> <em> <span> Texto
Definición

<strong> — Texto de gran importancia semántica. Se muestra en negrita.
<em> — Texto con énfasis. Se muestra en cursiva.
<span> — Contenedor en línea sin significado semántico propio. Se usa para aplicar estilos CSS a partes del texto sin crear salto de línea.

Sintaxis básica
<p>
  Texto con <strong>mucha importancia</strong>
  y con <em>énfasis</em>.
  Y con <span class="color-especial">color</span>.
</p>
Ejemplo práctico

Este párrafo tiene texto muy importante (strong) y texto enfatizado (em). También puedo usar span para colorear partes específicas.

✦ Buenas prácticas

Usa <strong> cuando el texto sea semánticamente importante (advertencias, datos clave), no solo para ponerlo en negrita visualmente. Para eso usa CSS con font-weight: bold. <span> es útil cuando necesitas aplicar una clase a una parte del texto sin añadir ningún significado extra.

<br> <hr> Texto
Definición

<br> — Inserta un salto de línea dentro de un párrafo sin crear un nuevo párrafo. Es una etiqueta vacía (no tiene cierre).
<hr> — Dibuja una línea horizontal que separa temáticamente dos partes del contenido. También es etiqueta vacía.

Sintaxis básica
<p>Primera línea.<br>Segunda línea en el mismo párrafo.</p>

<hr> <!-- línea divisoria -->
Ejemplo práctico

Primera línea del párrafo.
Segunda línea (usando <br>) en el mismo párrafo.

Este texto está separado por <hr>

✦ Buenas prácticas

Evita usar <br> para crear espacios entre párrafos, para eso usa margin con CSS. <br> tiene sentido en poemas, direcciones postales o cualquier texto donde el salto de línea forma parte del contenido mismo.
