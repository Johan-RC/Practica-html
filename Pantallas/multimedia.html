Multimedia

Etiquetas para insertar imágenes, audio, video y contenido externo directamente en la página, sin necesidad de plugins.

<img> Multimedia
Definición

Inserta una imagen en la página. Es una etiqueta vacía, no tiene etiqueta de cierre. El atributo src indica la ruta de la imagen y alt describe la imagen en texto (obligatorio para accesibilidad).

Sintaxis básica
<img
  src="ruta/imagen.jpg" <!-- dónde está la imagen -->
  alt="Descripción" <!-- texto alternativo (obligatorio) -->
  width="400" <!-- ancho en píxeles -->
  height="300" <!-- alto en píxeles -->
>
Ejemplo práctico
↑ Imagen con src externo, alt descriptivo, y clase CSS para el ancho
Uso adecuado

Para imágenes locales usa una ruta relativa: src="imagenes/foto.jpg". Para imágenes en línea usa la URL completa: src="https://...". Puedes controlar el tamaño con CSS usando la clase en lugar de los atributos width y height.

✦ Buenas prácticas

Siempre escribe un alt descriptivo y significativo. Si la imagen es solo decorativa, usa alt="" para que los lectores de pantalla la ignoren. Especifica siempre width y height para que la página no "salte" mientras carga. Usa loading="lazy" para imágenes que están abajo en la página.

<audio> <source> Multimedia
Definición

<audio> — Incrusta un reproductor de audio en la página.
<source> — Especifica el archivo de audio y su formato. Se usan varios <source> para garantizar compatibilidad: el navegador usa el primero que pueda reproducir.

Sintaxis básica
<!-- controls muestra los controles de reproducción -->
<audio controls>
  <!-- el navegador usa el primero que soporte -->
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
  <!-- mensaje si el navegador no soporta audio -->
  Tu navegador no soporta audio HTML5.
</audio>
Ejemplo práctico
↑ Reproductor de audio con controles nativos del navegador
✦ Buenas prácticas

Siempre usa controls para que el usuario pueda controlar la reproducción. Evita autoplay: puede molestar a los visitantes y algunos navegadores lo bloquean. Provee al menos formato MP3 para máxima compatibilidad entre navegadores.

<video> Multimedia
Definición

Incrusta un reproductor de video directamente en la página sin necesidad de instalar plugins externos. Funciona igual que <audio> pero para video.

Sintaxis básica
<!-- poster: imagen de portada antes de reproducir -->
<video controls width="640" poster="miniatura.jpg">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  Tu navegador no soporta video HTML5.
</video>
Ejemplo práctico
↑ Video con controls, poster (miniatura) y múltiples fuentes
✦ Buenas prácticas

Usa siempre poster para mostrar una imagen de portada antes de que el video cargue. Para mejorar la accesibilidad, agrega subtítulos usando la etiqueta <track>. Para videos muy pesados considera YouTube o Vimeo e incrústalos con un <iframe>.

<iframe> Multimedia
Definición

Incrusta un documento o página web externa dentro de la página actual. Se usa comúnmente para insertar mapas de Google Maps, videos de YouTube, formularios externos, y otros contenidos de terceros.

Sintaxis básica
<iframe
  src="https://url-del-contenido.com"
  width="600"
  height="400"
  title="Descripción del contenido" <!-- obligatorio para accesibilidad -->
  loading="lazy" <!-- carga solo cuando es visible -->
></iframe>
Ejemplo práctico
↑ Mapa de Bogotá incrustado con <iframe> desde OpenStreetMap
Uso adecuado

Para incrustar un video de YouTube: ve al video, haz clic en "Compartir" → "Insertar" y copia el código <iframe> que te dan. Para Google Maps: "Compartir" → "Insertar un mapa".

✦ Buenas prácticas

Siempre agrega el atributo title para que los lectores de pantalla describan qué hay dentro del iframe. Usa loading="lazy" para mejorar la velocidad. Solo incrusta contenido de fuentes de confianza; un iframe carga una página completa con su propio JavaScript.
