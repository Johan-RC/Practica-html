01 / 06 📐 Etiquetas Estructurales

Las etiquetas que dan la estructura base a una página web y le dan sentido semántico al contenido.

<header> <nav> <main> <section> <article> <footer>
02 / 06 ✏️ Texto y Contenido

Etiquetas para organizar y dar formato al texto de la página: títulos, párrafos y énfasis.

<h1>–<h6> <p> <strong> <em> <span> <br>
03 / 06 📋 Listas

Etiquetas para crear listas ordenadas, desordenadas y de definición para organizar información.

<ul> <ol> <li> <dl> <dt> <dd>
04 / 06 📝 Formularios

Etiquetas para crear formularios interactivos donde el usuario puede ingresar y enviar datos.

<form> <input> <label> <select> <button> <textarea>
05 / 06 🗂️ Tablas

Etiquetas para mostrar datos en forma de filas y columnas, como una hoja de cálculo.

<table> <thead> <tbody> <tr> <th> <td>
06 / 06 🎬 Multimedia

Etiquetas para insertar imágenes, audio, video y contenido externo en la página web.

<img> <audio> <video> <source> <iframe>
